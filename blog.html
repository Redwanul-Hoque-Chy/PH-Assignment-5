Question-1: What are the different ways to select an element in the DOM?

Five ways to select an element in the DOM. Those are:
1. getElementsByTagName
2. getElementById
3. getElementsByClassName 4. querySelector
5. querySelectorAll

Question-2: What is the difference between innerHTML, innerText, and textContent ?

innerHTML is taking all text in the node inluding html tag.

Question-3: What is event delegation in the DOM?

Event delegation is event that handle particular target.

Question-4: What is event bubbling in the DOM?

Event bubbling is event bubbles up to its parent and to root element.

Question-5: How do you create, add, and remove elements using JavaScript?

To create element method is document.createElement().
To add element method is element.appendChild();
To remove element method is element.remove();
